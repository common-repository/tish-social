== Description ==

[Premium](https://tishonator.com/plugins/tish-social) | [Demo](https://tishonator.com/plugins-demo/tish-social/)

Display latest Instagram images from ANY public Instagram account (without Access Token) by inserting a shortcode [instagram username="instagram_username"]. Responsive design, customizable, and easy to use plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `tish-social` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. insert the shortcode [instagram username="instagram_username"] to display Instagram latest images from account with username 'instagram_username' to any page or post
4. Enjoy!
